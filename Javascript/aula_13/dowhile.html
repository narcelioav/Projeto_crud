Estrutura de Loop do while

Do While executa a mesma função que while, mas ambos tem uma pequena diferença. O comando while verifica a variavel primeiro, e se ela for verdadeira, ele executa o codigo. Do while faz o contrario, ele executa o codigo primeiro e so depois verifica a variavel, se ela for verdadeira, então ele executa o codigo novamente.

do {
//codigo que sera executado
} while (comparação)
