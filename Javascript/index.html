O que é JavaScript

JavaScript é uma linguagem de programação.

Usado para manipular comportamentos na pagina.

O JavaScript trabalha com o HTML e CSS.

O JavaScript é uma linguagem de programação client-side (lado do cliente)

O JavaScript pode ser usado interno ou externo.

Segue um exemplo de uma função no JavaScript externo executada no Html atraves do comando onClick.

Variaveis

Variaveis são usadas para armazenar, recuperar e manipular valores que aparecem do codigo.

exemplo: var nome = "Nome";

String: quase qualquer valor entre espas simples ou aspas duplas;
Numerico: numero;
Booleano: true (verdadeiro) ou false (falso);

Exemplo de retorno de variaveis estaticas.

Constantes - Const

O const é usado para declarar variaveis constantes, o valor dessa variavel nunca é alterado, sempre é o mesmo e se você tentar alterar o valor nada acontecerá.

const nome = "nome";

Exemplo de variavel inalterado.

Operadores Matematicos

Os opeadores matematicos são utilizados para operações aritimeticas basicas

Operador Descrição
+ Adição
- Subtração
* Multiplicaçã
/ Divisão
% Modulo (resto da divisão)

Exemplo de operações matematicaas.

Operadores de Atribuição

Operadores de atribuição, atribuem um valor a uma variavel, e tornam o codigo mais eficiente.

Operador Descrição
= Atribuição simples
+= Atribuição com Adição
-= Atribuição com Subtração
*= Atribuição com Multiplicaçã
/= Atribuição com Divisão
%= Atribuição com Modulo (resto da divisão)

Exemplo de operações de atribuição.

Operadores de Comparação

Usado para comparar dois valores

Operador Descrição
== Igual a
!= Diferente de
< / < Menor que
> / > Maior que
<= / <= Menor ou igual a
>= Maior ou igual a

Exemplo de operações de comparação.

Operadores Logicos

Usado para verificar se os valores são verdadeiro ou falso.

Operador Nome Descrição
&& E Retorna TRUE se ambos os operadores forem verdadeiros.
|| OU Retorna TRUE se um ou outro forem verdadeiros.
! NÃO Retorna TRUE se o operadore não é verdadeiros.

Exemplo de operações logicos.

Operadores Incremento e Decremento

Usado para otimizarar a adção e subtração de valores.

Operador Descrição
++ Incrementa o valor do operador.
-- Decrementa o valor do operador.
